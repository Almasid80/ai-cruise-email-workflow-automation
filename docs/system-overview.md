# System Overview

This system automates the analysis of customer cruise emails and generates structured operational outputs.

The goal is to assist travel advisors by converting unstructured email messages into structured information that supports faster response preparation and workflow prioritization.

## Core Capabilities

- Email classification
- Urgency detection
- Customer intent identification
- Recommended operational next action
- AI-generated reply draft
- Spreadsheet-based workflow tracking

## Key Idea

Instead of manually interpreting every customer email, the system uses AI to produce structured operational insights that advisors can quickly review.

The automation assists advisors but keeps humans in control of final customer communication.
